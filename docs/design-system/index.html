Implementation Guide

How to use this design system in your Shopify theme.

Using Design Tokens

All design tokens are available as CSS custom properties. Include them in your theme's base CSS file to maintain consistency across all pages.

/* In your Shopify theme CSS */ .product-card { background: var(--color-surface); border-radius: var(--radius-lg); padding: var(--space-6); box-shadow: var(--shadow-md); } .product-title { font-family: var(--font-serif); font-size: var(--text-xl); color: var(--color-text); margin-bottom: var(--space-3); } .btn-add-to-cart { background: var(--color-primary); color: var(--color-white); padding: var(--space-3) var(--space-6); border-radius: var(--radius-md); transition: all var(--transition-base); }
