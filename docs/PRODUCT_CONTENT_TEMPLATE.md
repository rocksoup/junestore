## Product Title Guidelines

### Format
```
[Style Name] [Garment Type] in [Color/Pattern]
```

### Examples
- Lace Bralette in Black
- Satin Slip Dress in Blush Pink
- High-Waist Brief in Nude
- Floral Embroidered Teddy in Ivory
- Mesh Triangle Bra in Deep Red

### Rules
- Use title case for all words
- Include color/pattern name
- Be specific about garment type
- Keep under 60 characters for SEO
- Avoid vague terms like "Item" or "Product"

## Product Description Template

### Structure

Every product description should follow this three-paragraph structure:

1. **Opening (2-3 sentences):** Lead with the experience and feeling
2. **Details (2-4 sentences):** Describe materials, construction, and key features
3. **Versatility (1-2 sentences):** Suggest wearing occasions or styling options

### Opening Paragraph Template

Focus on the emotional experience and what makes this piece special:

```
[Product] [verb describing feeling/experience]. [Key design element] [creates/provides/delivers] [benefit], while [construction feature] [ensures/offers/guarantees] [comfort/support/fit benefit]. [One sentence about who this is for or when to wear it].
```

### Details Paragraph Template

Technical information presented in an engaging way:

```
Crafted from [fabric content with descriptive adjectives], this [garment type] features [construction detail 1] and [construction detail 2]. [Fabric stretch description] [creates/enables/provides] [fit benefit]. [Additional feature] adds [functional or aesthetic benefit].
```

### Versatility Paragraph Template

Practical styling and occasion suggestions:

```
Perfect for [occasion 1], [occasion 2], or [occasion 3]. [Styling suggestion] for [result/look]. [Care instruction highlight if relevant].
```

## Metafields Content Guide

### Fabric Content

**What to include:**
- Exact percentage composition
- Fabric type and quality descriptors
- Special properties (breathable, moisture-wicking, etc.)
- Lining information if applicable

**Format:**
```
[XX]% [Material], [XX]% [Material]
[Adjective] [texture/quality descriptor]
```

**Examples:**
```
82% Nylon, 18% Spandex
Soft, breathable mesh fabric with four-way stretch
```

```
100% Silk Charmeuse
Luxurious, lightweight fabric with natural temperature regulation
Lined with 95% Cotton, 5% Elastane for comfort
```

---

### Care Instructions

**What to include:**
- Washing method (hand wash/machine wash)
- Water temperature
- Drying method
- Special warnings (no bleach, no iron, etc.)
- Storage tip if relevant

**Format:**
```
[Washing instruction]
[Drying instruction]
[Prohibition 1]
[Prohibition 2]
[Optional: Storage tip]
```

**Examples:**
```
Hand wash cold with like colors
Lay flat to dry
Do not bleach or iron
Do not dry clean
```

```
Machine wash cold on delicate cycle in mesh bag
Hang to dry away from direct sunlight
Do not use fabric softener
Iron on low heat if needed
```

### Model Measurements

**What to include:**
- Size the model is wearing
- All four key measurements (bust, waist, hips, height)
- Consistent unit format

**JSON Format (Preferred):**
```json
{
  "size_worn": "M",
  "bust": "34\"",
  "waist": "26\"",
  "hips": "36\"",
  "height": "5'9\""
}
```

**Text Format (Alternative):**
```
Model is wearing size M
Bust: 34" | Waist: 26" | Hips: 36" | Height: 5'9"
```

**Measurement Guidelines:**
- Round to nearest inch
- Use straight inch marks (") not curly quotes
- Always include size worn first
- Use consistent order: bust, waist, hips, height

### Fabric Stretch

**What to include:**
- Type of stretch (two-way, four-way, moderate, etc.)
- How stretch affects fit
- Comfort or movement benefits

**Format:**
```
[Stretch type/degree] [for/provides] [benefit]
```

**Examples:**
```
Four-way stretch for flexibility and all-day comfort
```

```
Moderate stretch that forms to your body while maintaining structure
```

```
Light stretch with supportive compression
```

```
No stretch - maintains crisp shape and structure
```

**Stretch Levels:**
- **Four-way stretch:** Stretches in all directions, maximum flexibility
- **Two-way stretch:** Stretches in one direction (usually horizontal)
- **Moderate stretch:** Some give, forms to body
- **Light stretch:** Minimal give, mostly structured
- **No stretch:** Fully structured, maintains shape

---

### Recommended For

**What to include:**
- Body types this flatters (inclusive language)
- Occasions or use cases
- Specific benefits or features
- Support or comfort attributes
- 3-5 recommendations per product

**Format:**
```
[Body type/audience descriptor]
[Occasion/use case]
[Specific feature benefit]
```

**Examples:**
```
All body types
Everyday comfort
Special occasions
Support for larger busts
Petite-friendly sizing
```

```
Active lifestyles
Lounging and relaxation
Layering under clothing
Sensitive skin
Temperature regulation
```

**Inclusive Language Guidelines:**
- Use "all body types" rather than "universal fit"
- Focus on features (support, coverage) not body types as limitations
- Describe benefits, not restrictions
- Be positive and empowering
- Avoid comparative language (better for, ideal for)

## Writing Style Guide

### Brand Voice

June Lingerie's voice is:
- **Luxurious:** Elevated, quality-focused, sophisticated
- **Empowering:** Confident, body-positive, celebratory
- **Inclusive:** Welcoming to all bodies, identities, and experiences
- **Knowledgeable:** Expert without being condescending
- **Intimate:** Personal, warm, authentic

### Writing Rules

**DO:**
- Use sensory language (soft, smooth, luxurious)
- Write in second person ("you'll love") when appropriate
- Focus on benefits over features
- Be specific about materials and construction
- Use active voice
- Keep sentences concise and scannable

**DON'T:**
- Use clich√©s or generic marketing speak
- Make claims about "flattering" certain body types
- Use gendered assumptions
- Include sizes in descriptions (put in variants instead)
- Write long, dense paragraphs
- Overuse exclamation points

### SEO Keywords to Include

Naturally incorporate these terms where relevant:
- Lingerie, intimates, bralette, bra, underwear
- Fabric types: lace, satin, silk, mesh, cotton
- Comfort, support, coverage, fit
- Luxury, premium, quality
- Specific garment types

### Tone by Product Type

**Everyday Basics:**
- Focus: Comfort, quality, versatility
- Tone: Practical, trustworthy, straightforward

**Special Occasion:**
- Focus: Beauty, confidence, celebration
- Tone: Elevated, romantic, indulgent

**Loungewear:**
- Focus: Comfort, ease, relaxation
- Tone: Warm, cozy, inviting

**High-Performance (sport bras, etc.):**
- Focus: Support, movement, durability
- Tone: Active, confident, technical

## Quality Checklist

Before publishing any product content, verify:

### Product Description
- [ ] Follows three-paragraph structure (experience, details, versatility)
- [ ] Includes sensory and descriptive language
- [ ] Mentions key features and benefits
- [ ] Contains relevant SEO keywords naturally
- [ ] Written in brand voice (luxurious, empowering, inclusive)
- [ ] Free of typos and grammatical errors
- [ ] No gendered assumptions or exclusive language
- [ ] Length: 100-200 words

### Fabric Content
- [ ] Lists exact percentage composition
- [ ] Includes fabric quality descriptors
- [ ] Mentions special properties if applicable
- [ ] Notes lining if applicable

### Care Instructions
- [ ] Specifies washing method and temperature
- [ ] Includes drying instructions
- [ ] Lists relevant prohibitions
- [ ] Clear and actionable

### Model Measurements
- [ ] All five values present (size, bust, waist, hips, height)
- [ ] Proper formatting (straight quotes, consistent units)
- [ ] Valid JSON syntax if using JSON format
- [ ] Measurements are realistic and accurate

### Fabric Stretch
- [ ] Describes stretch type accurately
- [ ] Explains how stretch affects fit
- [ ] Under 100 characters
- [ ] Clear and informative

### Recommended For
- [ ] 3-5 items listed
- [ ] Inclusive, positive language
- [ ] Mix of body types and use cases
- [ ] Benefits-focused, not restrictive

### Overall
- [ ] Consistent tone across all fields
- [ ] Factually accurate
- [ ] Aligns with brand values
- [ ] Would help customer make informed purchase decision
